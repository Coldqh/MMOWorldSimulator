# MMOWorldSimulator 0.8.2 — Guild-Summoned World Bosses

## Что сделано

### 1) Guild-Summoned World Bosses
- добавлен гильдейский призыв мирового босса;
- призыв доступен только для ГМ / зама / офицеров;
- призыв работает только вне города, в текущей зоне / на текущем споте;
- у гильдии может быть только один активный гильдейский призыв одновременно;
- добавлен откат призыва: 12 игровых часов;
- призыв расходует:
  - Gold;
  - Raid Seals;
  - War Crests;
- после призыва автоматически создаётся рейд на босса;
- призывающий игрок автоматически вступает в рейд;
- участники из той же гильдии получают бонусные War Crests за убийство такого босса.

### 2) UI / Mobile Polish
- уменьшен общий мобильный UI;
- уменьшены topbar, bottom nav, panel padding, кнопки и карточки на телефонах;
- уменьшена ширина drawer на мобилках;
- исправлен sidebar player-card:
  - убран лишний квадрат-аватар в боковом меню;
  - карточка стала компактнее;
  - исправлен layout, чтобы GS-бейдж не ломал строку и ничего не вылезало.

### 3) World / Guild UI
- на экране Guild добавлен блок "Гильдейский призыв";
- показываются:
  - роль игрока;
  - откат;
  - активный призыв;
  - стоимость;
  - текущие Raid Seals / War Crests;
- на экране World мировые боссы теперь помечаются как:
  - обычный рейдовый мировой босс;
  - гильдейский призыв с именем гильдии.
