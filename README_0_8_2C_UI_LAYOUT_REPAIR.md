# 0.8.2c — UI Layout Repair

## Исправлено

- удалён глобальный Ultra Compact слой;
- восстановлены читаемые размеры текста и кнопок;
- панели снова занимают высоту по содержимому;
- карточные сетки больше не растягивают короткие элементы;
- инвентарь получает основную ширину экрана;
- экипировка получает фиксированную компактную колонку;
- названия и характеристики предметов могут занимать две строки;
- кнопка «Надеть» больше не сжимает название предмета;
- фильтры инвентаря прокручиваются горизонтально;
- на телефоне экипировка и инвентарь идут отдельными полноширинными блоками;
- боевой интерфейс не подвергается чрезмерному уменьшению;
- гильдейский призыв перенесён в отдельную вкладку «Призыв»;
- вкладка и содержимое призыва видны только ГМ;
- призыв больше не выводится под каждой вкладкой;
- кнопка выхода из гильдии выводится только в профиле;
- внутренние вкладки гильдии на телефоне прокручиваются горизонтально.
